## Introduction

Using "pdfman", you can edit your pdf like :- split, merge etc.

## Compatability

Python 3.6+ is required.

## Installation

```sh
pip install pdfman
```

## Usage

```sh
pdfman -h
pdfman merge file1.pdf file2.pdf
pdfman split file.pdf -r 2,4-6,8
```
